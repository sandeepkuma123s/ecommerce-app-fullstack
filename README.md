# 🛒 E-Commerce Full Stack Platform

A modern and responsive Full Stack E-Commerce application built using React, Node.js, Express, and MongoDB. The platform provides a complete online shopping experience with authentication, product management, order tracking, reviews, wishlist, cart management, and an admin dashboard.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/sandeepkuma123s/ecommerce-app-fullstack.git
cd ecommerce-app-fullstack
```

## Frontend Setup

```bash
cd client
npm install
npm run dev
```

## Backend Setup

```bash
cd server
npm install
npm run dev
```

# 🔐 Environment Variables

## Frontend (.env)

```env
VITE_API_URL=YOUR_BACKEND_URL/api
```

## Backend (.env)

```env
PORT=5000
MONGO_URI=YOUR_MONGODB_CONNECTION_STRING
JWT_SECRET=YOUR_SECRET_KEY
```
